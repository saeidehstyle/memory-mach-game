# Memory Game

This project is a simple **Memory Matching Game** built with **React**. The game challenges players to match pairs of cards by flipping them over. Each card has a front and a back, and when two cards match, they remain flipped.

## Features

- **Card Flip Animation**: Smooth animations when cards are flipped.
- **Match Logic**: Cards are matched based on their images.
- **Turns Count**: The number of turns taken is displayed.
- **New Game**: A button to shuffle the cards and start a new game.

## Technologies Used

- **React**: For building the user interface.
- **CSS**: For styling and animations.
- **JavaScript (ES6+)**: For game logic and interactivity.
- **Google Fonts**: Custom font imported for aesthetic purposes.
